06 · NON-NEGOTIABLES

UX SAFETY
01
渐进披露，不制造三套编辑器

黑／灰／白盒共用同一 Recipe；切换只改变控制深度，不丢参数。

02
状态不能只依赖颜色

颜色必须与文字、图标或形状共同表达；正文对比度达到 WCAG AA。

03
Agent 展示证据，不展示思维链

显示阶段、工具事件、预算、结果与失败原因；不使用角色群聊制造忙碌感。

04
不伪造进度与复现能力

Unknown 就写 Unknown；远程生成按 exact / seeded / recipe / record-only 分级。

05
稳定锁必须可解释

显示锁定维度、作用域、继承与冲突；模型不支持时明确降级。

06
拒绝 AI 视觉套路

无紫蓝渐变、无泛滥玻璃拟态、无大圆角卡片海、无装饰性 Agent 头像。
